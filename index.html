DS Universe - Cloud-Based Dataset Repo

I built a cloud-native platform that enables users to contribute, manage, and access datasets—think of it as GitHub, but for data. The application was architected using AWS services like EC2, S3, RDS (MySQL), IAM, and Route 53, ensuring seamless scalability, strong security, and high availability. I implemented an elastic load balancer for fault tolerance and leveraged AWS IAM to enforce secure access controls. To enhance performance, I tuned database queries and added indexing strategies optimized for large datasets. All infrastructure was managed and automated through AWS Cloud Shell, streamlining deployment and resource provisioning.

I developed a real-time collaborative note-taking web app that enables seamless multi-user editing using WebSockets (Socket.io). The frontend is deployed on Vercel for fast, global access, while the backend runs on AWS EC2 for scalability and control. I designed and implemented a RESTful API with Express.js and MongoDB, focusing on efficient CRUD operations and performance-tuned queries to enhance document retrieval speed. StockFlow - Real-Time Stock Market Data Pipeline

I engineered a real-time stock market data pipeline using Kafka Producers and Consumers, enabling high-throughput data ingestion with fault tolerance via a distributed Kafka cluster hosted on AWS EC2. The system was designed for scalability and cost-efficiency, leveraging AWS S3, Glue, and Athena for serverless data processing and querying. By automating ETL workflows with AWS Lambda, I streamlined data transformation and significantly improved performance for large-scale financial datasets.
